# EC業務ナレッジ チャットボット 月額コスト試算 — 300名シナリオ

> **想定**: EC全体展開（約300名利用）  
> 単価は 2026年2月時点の Azure OpenAI 料金に基づく

## 現状・6万円シナリオとの変更点

| 項目 | 現状（30名） | 6万円（150名） | **本シナリオ（300名）** |
|---|---|---|---|
| 利用者数 | 30人 | 150人 | **300人** |
| 1日あたり質問数 | 20件/日 | 25件/日 | **25件/日** |
| Azure AI Search | Basic ($73) | Basic ($73) | **Standard S0 ($250)** |
| App Service Plan | B1 ($13) | S1 ($50) | **S2 ($100)** |
| Azure SQL Database | Basic 5DTU ($4.90) | S1 ($30) | **S2 ($75)** |
| ナレッジ更新 | 週次・4資料/回 | 同左 | 同左（変更なし） |

> ⚠️ 300名規模では Azure AI Search を **Basic → Standard S0** へアップグレードが必要。  
> これがコスト増の主要因（+¥26,550/月）。
